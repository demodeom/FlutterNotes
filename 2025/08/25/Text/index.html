文字显示和样式

组件 Text

基础使用

Style 样式

文本样式通过属性 style 设置，值类型 TextStyle

常用属性

  • 文字大小
  • 文字家族
  • 文字颜色
  • 文字粗细
  • 文字样式（倾斜）

TextStyle 类中属性

属性名 属性值类型 补充
fontSize double 设置字体大小
fontFamily String 设置字体家族
color Color 设置文字颜色
fontWeight FontWeight 设置字体粗细，值越大，文字越粗
FontWeight.w100
FontWeight.w200
FontWeight.w300
FontWeight.w400
FontWeight.w500
FontWeight.w600
FontWeight.w700
FontWeight.w800
FontWeight.w900
FontWeight.normal 等价于 FontWeight.w400
FontWeight.bold等价于 FontWeight.w700
fontStyle FontStyle 设置字体样式
FontStyle.normal 正常(默认值)
FontStyle.italic倾斜

文字装饰线

TextStyle 类中属性

属性名 属性值类型 补充
decoration TextDecoration 设置文字装饰线位置
TextDecoration.lineThrough 中划线
TextDecoration.underline下划线
TextDecoration.overline 上划线
decorationColor Color 设置文字装饰线的颜色
decorationStyle TextDecorationStyle 设置文字装饰线的样式
TextDecorationStyle.dotted 点划线
TextDecorationStyle.dashed 虚线
TextDecorationStyle.double双实线
TextDecorationStyle.solid实现
TextDecorationStyle.wavy 波浪线
decorationThickness double 设置字体装饰线粗细

字母、单词间距

TextStyle 类中属性

属性名 属性值类型 补充
letterSpacing double 设置字母之间的间距
wordSpacing double 设置单词之间的间距

使用 letterSpacing 设置汉字之间的间距，wordSpacing 属性对汉字无效

背景颜色、阴影

TextStyle 类中属性

属性名 属性值类型 补充
backgroundColor Color 设置文字背景颜色
shadows List<Shadow> 设置文字阴影

shadows 设置文字阴影，文字可以设置多个方向的阴影， Shadow 属性

属性名 属性值类型 补充
color Color 设置文字背景颜色
offset Offset(double dx, double dy) 设置文字阴影方向
dx 表示水平方向偏移，向右为正，向左为负
dy 表示垂直方向偏移，向下为正，向上为负
blurRadius double 设置文字阴影模糊值
值越大，阴影越模糊

自定义字体

以 霸道总裁 字体为例， 字体下载地址 https://zitiku.org/font/53

字体存储位置

字体存储位置可以自定义，比如：在项目根目录创建目录 asset/fonts, 将下载字体存放到 asset/fonts 目录
