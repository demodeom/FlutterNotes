### Row、Column

`Row` 将多个子元素水平显示

`Column` 将多个子元素垂直显示

### Row

```dart
Row(
  children: [
    Container(
      width: 50,
      height: 50,
      decoration: BoxDecoration(
        color: Colors.red,
      ),
    ),
    Container(
      width: 50,
      height: 50,
      decoration: BoxDecoration(
        color: Colors.orange,
      ),
    ),
    Container(
      width: 50,
      height: 50,
      decoration: BoxDecoration(
        color: Colors.yellow,
      ),
    ),
    Container(
      width: 50,
      height: 50,
      decoration: BoxDecoration(
        color: Colors.green,
      ),
    ),

      height: 50,

        color: Colors.cyan,
      ),

      width: 50,

      ),
    ),
    Container(
      width: 50,

      ),

  ],
```
